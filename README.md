## Description

POC AWS SQS with [Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ pnpm install
```

## Before running the app

```bash
  - create your own .env file in the root directory
  - you can follow the example from .env.example
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
